# 《单路猎人：无限传送带》小游戏设计文档 v0.1

> 关键词：LOL 灵感、无限传送带、横移射击、补刀经济、QWER 技能、装备合成、野怪资源、元素龙、Boss、5 分钟一局。

---

## 0. 一句话概念

玩家操控一名英雄站在单路防线前，只能左右横移、普攻和释放技能。怪物、小兵、炮车、宝箱、装备、野怪、元素龙和 Boss 会从无限传送带上源源不断出现。玩家通过补刀赚钱、升级技能、合成装备、抢资源和打 Boss，在 5 分钟内守住己方水晶并推爆敌方核心。

---

## 1. 设计目标

### 1.1 核心体验

- **像 LOL 清线一样上头**：补刀、清兵、躲技能、抢资源、装备成型。
- **像割草游戏一样爽**：局内成长快，技能越来越夸张，后期满屏弹道和特效。
- **像开盲盒一样刺激**：传送带上会刷怪、宝箱、装备、陷阱、Boss，玩家需要不断做取舍。
- **像 Roguelike 一样可重玩**：每局技能、装备、Boss、元素事件、掉落组合都不同。

### 1.2 单局时长

- MVP 单局：**3-5 分钟**。
- 完整版单局：**5-8 分钟**。
- 失败原因：玩家死亡、己方水晶被破、最终核心未能限时击破。
- 胜利目标：击败最终 Boss 或推爆敌方水晶核心。

### 1.3 目标平台

优先建议：

- Web：TypeScript + Phaser 3。
- 移动端：横屏 H5 / 小程序。
- 后续可迁移：Unity / Godot。

## 2. 核心画面与空间布局

### 2.1 推荐视角

采用 **2D 俯视 / 2.5D 斜俯视**。

屏幕结构：

```text
┌──────────────────────────────┐
│ 敌方核心 / Boss 出口          │
│                              │
│  传送带轨道区域               │
│  怪物、宝箱、装备、陷阱向下推进 │
│                              │
│                              │
│        玩家横移区域            │
│ ← 玩家只能左右移动 →           │
│                              │
│ 己方水晶 / 防御塔血条          │
└──────────────────────────────┘
```

### 2.2 传送带轨道

建议使用 **5 条轨道**。

| 轨道 | 用途 |
|---|---|
| Lane 0 | 左外侧，适合刷资源、侧翼怪、陷阱 |
| Lane 1 | 左中路，普通兵线 |
| Lane 2 | 中路，炮车、Boss、宝箱高频出现 |
| Lane 3 | 右中路，普通兵线 |
| Lane 4 | 右外侧，适合刷资源、刺客怪、野怪入口 |

### 2.3 对象移动方向

- 怪物、宝箱、装备、陷阱从屏幕上方向下移动。
- 玩家位于屏幕下方，只能左右移动。
- 敌人越过玩家区域后，会对己方水晶造成伤害。
- 宝箱、材料、资源如果离开屏幕，则视为错过。

## 4. 核心循环

```text
怪物/宝箱/装备从传送带出现
        ↓
玩家左右横移选择目标
        ↓
普攻与技能清怪、补刀、开箱、抢资源
        ↓
获得金币、经验、装备材料、技能升级
        ↓
三选一强化 / 装备合成 / 获得 Buff
        ↓
出现精英怪、元素龙、Boss、敌方核心
        ↓
胜利或死亡
        ↓
结算本局流派与最高输出组合
```

## 5. 胜负条件

### 5.1 胜利条件

MVP 版本可使用以下任意一种：

1. 存活 5 分钟，并击败最终 Boss。
2. 在 5 分钟内推爆敌方核心。
3. 击败 3 个元素 Boss 后召唤最终核心，击破后胜利。

### 5.2 失败条件

| 失败条件 | 说明 |
|---|---|
| 玩家生命归零 | 被怪物、技能、陷阱、Boss 击杀 |
| 己方水晶归零 | 漏掉太多兵线或炮车 |
| 时间耗尽 | 未能击破最终核心 |

## 7. 传送带对象类型

### 7.1 总分类

| 类型 | 例子 | 交互方式 |
|---|---|---|
| 小兵 | 近战兵、远程兵、炮车兵 | 击杀、补刀 |
| 精英怪 | 盾兵、刺客、法师、辅助怪 | 优先击杀或躲避技能 |
| 宝箱 | 普通箱、金箱、诅咒箱、宝箱怪 | 攻击打开 |
| 装备材料 | 短剑、法杖、护甲、宝石 | 拾取或攻击获得 |
| 野区资源 | 红 Buff、蓝 Buff、河道视野怪 | 限时抢夺 |
| 陷阱 | 炸弹、减速泥潭、沉默符文 | 躲避或打爆 |
| Boss | 元素龙、大龙、超级兵王、敌方核心 | 多阶段战斗 |

### 7.2 生命周期

每个传送带对象都有统一生命周期：

```text
spawn → enter_screen → active → interact → resolved / escaped
```

| 阶段 | 说明 |
|---|---|
| spawn | 由刷怪导演生成，决定轨道、速度、类型、稀有度 |
| enter_screen | 出现在屏幕上方，有短暂预警 |
| active | 可以被锁定、攻击、拾取、碰撞 |
| interact | 被击杀、打开、触发、吸收、爆炸 |
| escaped | 离开玩家防线，造成漏刀、扣塔血或错失奖励 |

## 9. 战斗系统

### 9.1 自动普攻

推荐自动攻击规则：

1. 优先攻击玩家正前方同轨道目标。
2. 同轨道无目标时，攻击最近目标。
3. 如果存在“锁定优先级”目标，优先攻击：Boss > 炮车 > 刺客 > 宝箱 > 普通兵。
4. 玩家释放技能时，可以覆盖普攻锁定。

### 9.2 补刀系统

补刀是本游戏最重要的 LOL 感来源。

| 击杀结果 | 条件 | 奖励 |
|---|---|---|
| 完美补刀 | 玩家最后一击击杀，且目标处于补刀提示窗口 | 金币 + 经验 + 连击 |
| 普通击杀 | 玩家击杀但不在窗口 | 经验 + 少量金币 |
| 助攻击杀 | 召唤物或持续伤害击杀 | 经验 |
| 漏刀 | 目标离开传送带底部 | 扣己方水晶血，断连击 |

补刀提示：

- 目标血量低于玩家普攻伤害时，血条边框变亮。
- 连续补刀会触发音效和 UI 连击数字。
- 连续 10 次补刀进入“专注状态”：攻击速度 +20%，持续 5 秒。

### 9.3 伤害类型

| 类型 | 说明 | 克制关系 |
|---|---|---|
| 物理 | 普攻、飞刀、箭矢 | 对低护甲目标强 |
| 法术 | 火球、雷电、奥术波 | 对高护甲目标有效 |
| 真实 | 斩杀、点燃、Boss 奖励 | 无视护甲魔抗 |
| 持续 | 燃烧、中毒、流血 | 适合处理高血量目标 |
| 爆发 | 暴击、大招、斩杀 | 适合抢补刀和秒精英 |

### 9.4 怪物职责体系

为了做出“团战感”，敌人不只是血量不同，而是有职责。

| 职责 | 行为 | 玩家决策 |
|---|---|---|
| 坦克 | 护盾高，挡在前排 | 先破盾或绕开 |
| 射手 | 持续射击玩家 | 快速击杀 |
| 法师 | 蓄力范围技能 | 看红圈横移躲避 |
| 刺客 | 突进到玩家附近 | 留 E 或闪现反制 |
| 辅助 | 给怪物加盾回血 | 优先秒掉 |
| 召唤 | 不断生成小兵 | 尽快打断 |

## 13. 局内升级：三选一强化

### 13.1 触发方式

- 升级时触发。
- 打开金宝箱触发。
- 击败元素 Boss 触发。
- 连续补刀达到阈值触发小强化。

### 13.2 强化类型

| 类型 | 示例 |
|---|---|
| 属性强化 | 攻击力 +15%、攻速 +10%、生命 +30 |
| 技能强化 | Q 穿透 +1、W 护盾持续时间 +1 秒、E CD -20% |
| 普攻强化 | 普攻弹射、普攻附带燃烧、普攻斩杀低血目标 |

## 20. 推塔与最终核心

### 20.1 推塔阶段

击败小 Boss 或大龙后，传送带方向可以短暂反转，表现为玩家开始反推。

```text
正常阶段：敌人从上往下压过来
强推阶段：玩家火力压制，敌方防御塔/核心从上方进入屏幕
```

### 20.2 敌方防御塔

| 机制 | 说明 |
|---|---|
| 防御塔血量 | 需要持续输出才能打掉 |
| 塔弹 | 周期性锁定玩家轨道发射高伤害弹 |
| 塔盾 | 每隔一段时间生成护盾，需要击杀小兵破盾 |
| 镀层奖励 | 打掉一层塔皮给金币和装备材料 |

### 20.3 敌方水晶核心

最终 Boss 设计：

- 核心本体固定在屏幕上方。
- 周围有 3 个水晶节点。
- 节点会分别召唤小兵、发射弹幕、给核心回血。
- 玩家需要先破节点，再输出核心。
- 5 分钟结束前击破核心即胜利。

## 21. 流派设计

### 21.1 暴击补刀流

关键词：暴击、攻速、补刀、金币滚雪球。

核心装备：无尽之刃型、反曲弓、暴击手套。  
核心强化：连续补刀增加暴击率，暴击击杀返还 Q 冷却。

### 21.2 法术清屏流

关键词：技能强度、冷却缩减、范围伤害。

核心装备：卢登回声型、冰霜权杖型、蓝水晶。  
核心强化：技能命中弹射、技能击杀爆炸、R 冷却降低。

### 21.3 攻速吸血流

关键词：高频普攻、生命偷取、站桩输出。

核心装备：破败之刃型、饮血剑型、反曲弓。  
核心强化：普攻附带当前生命伤害，吸血溢出护盾。

### 21.4 坦克反伤流

关键词：护盾、减伤、反伤、火焰光环。

核心装备：日炎铠甲型、复活甲型、生命宝石。  
核心强化：受击反弹、护盾爆炸、低血量减伤。

### 21.5 刺客斩杀流

关键词：横移闪避、低血量斩杀、爆发窗口。

核心装备：长剑、暴击手套、暗影魂。  
核心强化：完美闪避后下一击暴击，斩杀返还 E 冷却。

### 21.6 召唤推线流

关键词：召唤物、炮台、友方小兵、推塔。

核心装备：召唤核心、冷却齿轮、蓝水晶。  
核心强化：每 10 秒召唤友军，召唤物继承攻速和暴击。

## 35. 最终建议

第一版不要追求复杂。先把下面这个闭环做爽：

```text
五条传送带 → 小兵下压 → 玩家横移自动攻击 → 补刀给金币 → 升级三选一 → 开金箱出装备 → 火龙 Boss → 最终水晶核心
```

只要 **补刀反馈、宝箱惊喜、装备质变、Boss 压迫** 做出来，这个小游戏就会很容易上头。
